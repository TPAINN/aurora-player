Open Source

Built for everyone.
Owned by no one.

Aurora runs entirely on free APIs. No subscription. No token quota. No server that can break your experience. Thousands of users, zero limits.

●
LRCLib — primary lyric source
Open database. Word-level timestamps on most tracks. Zero cost.
●
YouTube IFrame API — zero cost playback
No API key. No quota. Streams the highest available audio quality.
●
iTunes Search API — free metadata
Album art, artist info, duration — no auth, unlimited requests.
●
MIT License — forever open
Fork it, host it, modify it. The code belongs to everyone.

How It Works

Three steps to the full experience.

01

Search any song

Type artist and track. Aurora queries iTunes, Genius, and YouTube simultaneously. Best match found in under a second.

02

Aurora analyzes

Album art colors extracted live. Synced lyrics load from LRCLib. BPM computed in real-time from the audio stream.

03

Feel the song

Every word highlights on cue. Background breathes with the beat. Chorus entries burst with expanding energy rings.
